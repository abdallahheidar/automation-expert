Project 2

This tool uses Excel VBA to automate the process of analyzing employee performance and work hours data. Users simply need to select the CSV files containing the data and click a button. The tool will then combine the data into a single master sheet, combine the performance/working hours for all days, remove all duplicate records, and highlight all employees who worked more than 8 hours per day and on which days.

This tool can be used by businesses of all sizes to improve employee performance and productivity. It can also be used by HR professionals to track employee performance and identify areas where improvement is needed.

  • Date: September 2023
  • Service: Excel Reporting Automation

Project 3

I successfully developed a Telegram bot with advanced monitoring features for securities, including stocks and cryptocurrencies. The bot utilized the TradingView API to gather real-time pricing data, enabling users to stay updated on market trends and make informed decisions.

Key Features: Telegram Bot Interface: Designed and implemented a user-friendly Telegram bot interface for seamless interaction. Securities Monitoring: Integrated the bot with the TradingView API to monitor a wide range of securities, including stocks and cryptocurrencies. Real-Time Pricing: Leveraged the TradingView API to fetch and display current pricing information for the monitored securities. Summary Generation: Developed a summary feature that provided users with buy/sell recommendations, current prices, and the Relative Strength Index (RSI) for each security. Alert System: Implemented an alarm mechanism, allowing users to receive regular updates and notifications on the monitored securities. Outcome: Through this project, I successfully created a robust Telegram bot that empowered users with real-time market insights. The bot's comprehensive features, including summary generation and the alert system, provided users with valuable information for making informed investment decisions. Technologies Used: Telegram Bot API TradingView API Python programming language This project showcases my expertise in developing Telegram bots, integrating APIs, and working with financial data. It highlights my ability to deliver user-friendly solutions that provide real-time market data and insights to support investment decisions.
